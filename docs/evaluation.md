# Evaluation Module

## Implemented

- Retrieval metrics: Recall@k, Hit@k, MRR
- Generation metrics: token overlap precision/recall/F1, citation precision/recall
- Markdown summary reports and per-query CSV detail exports
- Simple ablation comparison report over saved evaluation summaries

## Experimental

- Minimal smoke gold labels
- Token-overlap generation metrics for extractive baseline

## Planned

- Larger benchmark datasets
- nDCG and graded relevance
- LLM-based answer quality evaluation
- Automated experiment registry integration
