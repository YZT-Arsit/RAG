# Retrieval Module

## Implemented

- BM25 retriever
- Dense baseline retriever
- Hybrid score fusion
- Retrieval result JSONL export

## Experimental

- Dense baseline based on character n-gram TF-IDF similarity
- Score-normalized hybrid fusion without reranking

## Planned

- Real embedding-based dense retrieval
- ANN index
- Reranker
- Retrieval evaluation against qrels
